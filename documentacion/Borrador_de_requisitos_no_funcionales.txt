					Borrador de requisitos no funcionales

RNF-001: Temporización

El sistema tiene que dar una respuesta en menos de 2 microsegundos.

RNF-002: Disponibilidad

El sistema debe estar disponible para su uso las 24 horas del día durante todo el año.

RNF-003: Compatibilidad

El sistema debe ser compatible con los navegadores Mozilla Firefox, Google Chrome e Internet Explorer.

RNF-004: Usabilidad

El sistema tiene que disponer de un acceso adecuado a los participantes, con una interfaz cómoda y sencilla. 

RNF-005: Seguridad

Cada usuario contará con un nombre de usuario y una clave para acceder al sistema.
